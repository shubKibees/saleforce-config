How to configure UTMS.
  • You have to add hidden input to the form, 
<input type="hidden" id="utm_campaign" name="utm_campaign" value="">
<input type="hidden" id="utm_medium" name="utm_medium" value="">
<input type="hidden" id="utm_source" name="utm_source" value="">
  • Now you have to fill these input, by taking parameter value from url of that page. Example: late say you have a url with utm like given below https://cloud.connect.canon.co.in/mfd/?utm_source=Google&utm_medium=CPC&utm_campaign=Search+ad&utm_term=+
  • put value of utm parameter into those hidden input, using code given bellow.
function setUTM() {
         const utmCampaign = getURLParameter('utm_campaign');
         const utmMedium = getURLParameter('utm_medium');
         const utmSource = getURLParameter('utm_source');
         if (utmMedium && utmSource) {
            if (utmCampaign) {
               document.getElementById('utm_campaign').value = utmCampaign;
            } else {
               document.getElementById('utm_campaign').value = "";
            }
            document.getElementById('utm_medium').value = utmMedium;
            document.getElementById('utm_source').value = utmSource;
         }
      }
setUTM();
  • at the end, create a input field in database for utm

Validation of form.
  • just add the below code, and add an extra parameter if you have in the form
 
      function validateData() {
         var nameformate = /^[a-zA-Z\s]*$/;
         var mailformat = /^\w+([\.-]?\w+)*@\w+([\.-]?\w+)*(\.\w{2,})+$/;
         var phoneformat = /^\d{10}$/;

         var name = document.getElementById('name').value;
         var mobile = document.getElementById('mobile').value;
         var state = document.getElementById('state').value;
         var email = document.getElementById('email').value;
         var city = document.getElementById('city').value;
         var authoriseCheckbox = document.getElementById('authorise');
         var newslettersCheckbox = document.getElementById('newsletters');

         var isValid = true;
         var errorMessage = '';

         switch (true) {
            case name.length == 0 || !name.match(nameformate):
               errorMessage += "Please enter a valid name.\n";
               isValid = false;
               break;
            case mobile.length == 0:
               errorMessage += "Please enter a mobile number.\n";
               isValid = false;
               break;
            case !mobile.match(phoneformat):
               errorMessage += "Please enter a valid 10-digit mobile number.\n";
               isValid = false;
               break;
            case state.length == 0:
               errorMessage += "Please select a state.\n";
               isValid = false;
               break;
            case email.length == 0:
               errorMessage += "Please enter an email address.\n";
               isValid = false;
               break;
            case !email.match(mailformat):
               errorMessage += "Please enter a valid email address.\n";
               isValid = false;
               break;
            case city.length == 0 || !city.match(nameformate):
               errorMessage += "Please enter a valid city.\n";
               isValid = false;
               break;
            case !authoriseCheckbox.checked:
               errorMessage += "Please agree to the terms and conditions.\n";
               isValid = false;
               break;
            case !newslettersCheckbox.checked:
               errorMessage += "Please subscribe to newsletters.\n";
               isValid = false;
               break;
         }

         if (!isValid) {
            return errorMessage;
         } else {
            return 1;
         }

         return isValid;
      }

State and city implementation

  • also add ID in the form in HTML, like given below
<select class="form-control" aria-label="form-field" name="state" id="state" required>
                        <option value="">Select State</option>
                        </select>
<select class="form-control" aria-label="form-field" aria-label="form-field" name="city" id="city" required="">
                           <option value="">Select City</option>
                        </select>

  • we have add id=”state” and id=”city” to the form field, using that our statecity.js script will recognize the position where state and city are need to add
